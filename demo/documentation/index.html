Favicon & Logo


Replace the favicon.ico with the .ico file that you created for your website. Favicon.ico is located in the "image" folder, located at the root folder. Don't forget to change the icon.png for retina devices.

To change the logo, go to the "image > logo" folder and replace the logo.png with your amazing brand logo.

HTML Structure


The html template uses the Bootstrap v5.0 with valid HTML5 tags. This theme features a responsive layout that incorporates support for four columns. All content within the content area is nested within specific classes, each of which is pre-defined.

CSS Files and Structure


Mainly three main CSS files are used in this theme. The first one bootstrap.css, second one is style.css and third one is responsive.css to control responsive layouts.

CSS and HTML files' each sections are labeled. If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style classes that needs to be edited.

Jquery and javascript


This theme imports three types of Javascript files.

  • jQuery
  • Custom scripts (script.js)
  • Some plugins:
    • Isotope Masonry
    • Counter Up
    • Waypoint JS
    • Maginific Popup
    • Nice Select
    • Swiper JS

  • jQuery is a Javascript library that greatly reduces the amount of code that you must write.

In addition to the custom scripts, We have implemented a few plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file. The only necessary thing to know is how to call the method, you can find documentation for them on their official websites.
